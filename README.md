# flower-classification
Start template for classifying flowers with data


## Need to have
* Python 3
* pip


# Get started

### (Optinal) Create Virtual env
```
python -m venv venv
```
### Install deps
```
pip install -r requirements.txt
```

# Scripts

### (Optional) Check all modules are installed
```
python check_deps.py
```

### Run demo
```
python main.py
```
